## 📌 Sobre o Projeto

Este projeto é uma **calculadora de consumo de energia elétrica**, desenvolvida em Python, que permite estimar o gasto mensal de aparelhos eletrônicos com base no tempo de uso diário.

💡 Ideal para conscientização sobre consumo de energia e economia!

---

## 🎯 Objetivo

Auxiliar usuários a entenderem o consumo de energia dos seus aparelhos e estimar o custo mensal, contribuindo para um uso mais consciente da eletricidade.

---

## 🎯 Funcionalidades

- ✅ Entrada do nome do aparelho  
- ✅ Entrada da potência (Watts)  
- ✅ Entrada do tempo de uso diário (horas)  
- ✅ Cálculo automático do consumo mensal (kWh)  
- ✅ Estimativa de custo mensal  

---

## 🧮 Fórmula Utilizada

```bash
consumoMensal = (potencia * horasDia * 30) / 1000

💰 Cálculo de Custo
custo = consumoMensal * 0.75

(Valor do kWh estimado em R$ 0,75)

▶️ Como Executar
Clone o repositório:
git clone https://github.com/seu-usuario/consumo-energia.git
Acesse a pasta do projeto:
cd consumo-energia
Execute o programa:
python app.py
📷 Exemplo de Uso
=== Calculadora de Consumo de Energia ===

Digite o nome do aparelho: Geladeira
Digite a potência (em watts): 150
Digite o uso diário (em horas): 10

=== Resultado ===
Aparelho: Geladeira
Consumo estimado: 45.00 kWh/mês
Custo estimado: R$ 33.75
